# Requirements Archive: v1.0 — Working PDF → KMZ Converter

**Defined:** 2026-05-12 · **Archived:** 2026-06-05 (milestone closed as-is)
**Core Value:** Accurately extract post data from INFOVIAS PDF files and produce a georeferenced KMZ file.

> Outcome legend: ✅ delivered · 🟡 delivered with known limits (carried to v1.1) · ⏸ deferred to v1.1

## v1 Requirements — Final Status

### PDF Parsing

- [x] **PDF-01** — Parse INFOVIAS-format PDFs in-browser — ✅ (pdf.js + OCG layer filtering)
- [x] **PDF-02** — Extract post/pole identifiers and types — ✅ (OCR via Tesseract.js; numbers are vector paths)
- [x] **PDF-03** — Extract inter-post distances — ✅ (`Distância_Poste` layer)
- [x] **PDF-04** — Extract post x,y drawing positions — ✅ (Poste-symbol / layer-0 centroids)
