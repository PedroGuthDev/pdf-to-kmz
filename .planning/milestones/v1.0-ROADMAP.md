## Overview

A client-side, single-page web app that converts INFOVIAS/FTTH fiber-infrastructure
PDF project files into georeferenced KMZ for Google Earth. The user uploads a PDF,
supplies the GPS of the first post, and the tool OCRs post numbers, extracts the
drawing layout, calculates per-post coordinates via per-page UTM-grid calibration
(with an optional DWG/DXF region-pairing graph-walk path for higher accuracy), and
emits a customizable KMZ. v1.0 proved the full pipeline end-to-end on multiple real
routes; what remains (broad cross-PDF generalization, per-route accuracy rework,
formal UI polish) is carried into v1.1.

## Milestone Summary

**Key Decisions (validated):**

- Client-side only (no backend) — pdf.js + Tesseract.js via CDN run fully in-browser.
- OCR via Tesseract.js per-page crop — post numbers are vector paths in `Numero_Poste`, so text extraction is impossible; 2× canvas crop + digit whitelist + pageseg 7.
- Per-page UTM-grid calibration (not sequential GPS chaining) — drawing layout + grid gives absolute placement.
- Poste-symbol canonical (x,y) (D-ACC-10) — label+cable-aware match, not cable vertices.
- DWG/DXF region-pairing graph-walk as a higher-accuracy path with transparent PDF-only fallback.
- Hack-free generalization (260601-k1a → 260602-decouple) — replaced hardcoded post-number guards with geometry/topology predicates; junction-from-DWG-region-geometry landed.

**Issues Resolved:**

- Broken text-proximity post-number extraction → OCR pipeline (Phase 1).
- Sequential-chaining drift → UTM-grid calibration (Phase 2).
- Siriu DWG graph-walk: connections-pruning/walk coupling fixed via walkConnections snapshot; gate passes all 85 posts.
- Label mis-association decoupled from the graph-walker (phantom load-bearing edges removed; all 4 gates green).

**Issues Deferred / Technical Debt (carried to v1.1 — see STATE.md Deferred Items):**

- **LC posts-1–20 coordinated post-positioning + calibration rework** — own milestone scope (`260603-n4k-MILESTONE-SCOPE.md`); Phases 1 + 1.5 (LC + Siriu per-post position gates) shipped, Phases 2–5 pending.
