<files>parser/kml-color.js</files>
<action>
Create `parser/kml-color.js` with named exports only (no default export).

Export `hexToKmlColor(hex, alpha = 0xff)`:
- Accept `#RRGGBB` or `RRGGBB` (strip `#`, require exactly 6 hex digits).
- Return lowercase KML color string `aabbggrr` where alpha is two hex digits from `alpha` (0–255).
- Example: `hexToKmlColor('#ff0000')` → `'ff0000ff'` (opaque red).
- Throw `Error` with message containing `invalid hex` if input invalid.

Optional export `presetToKmlColor(presetKey)` if `kmz-defaults.js` needs it — may stay in defaults file instead.
</action>
<acceptance_criteria>
- `parser/kml-color.js` exports `hexToKmlColor`
- `hexToKmlColor('#ff0000') === 'ff0000ff'`
- `hexToKmlColor('#0000ff') === 'ffff0000'` (blue in KML byte order)
- `hexToKmlColor('00ff00', 128)` starts with `'80'` alpha prefix
- Invalid input throws Error
- No `export default`
</acceptance_criteria>
</task>

<action>
Create `parser/kmz-defaults.js` importing `hexToKmlColor` from `./kml-color.js`.

Export `DEFAULT_OPTIONS` object with keys:
- `iconHref`: `'http://maps.google.com/mapfiles/kml/shapes/placemark_square.png'` (comment: verify visually in Earth per D-IC-02)
- `iconColor`: preset key string e.g. `'green'`
- `lineColor`: preset key e.g. `'red'`
- `lineWidth`: number `3`
- `labelColor`: preset key e.g. `'white'`
- `labelScale`: number `1`
- `lineDescription`: string `''`

Export `PRESET_COLORS` map: at least `red`, `green`, `blue`, `white`, `yellow`, `black` → `#RRGGBB` hex strings.

Export `mergeOptions(user = {})` — shallow merge `{ ...DEFAULT_OPTIONS, ...user }`.

Export `resolveStyleColors(merged)` returning `{ iconColorKml, lineColorKml, labelColorKml }` as `aabbggrr` strings via preset lookup + `hexToKmlColor`. Unknown preset keys fall back to green/red/white respectively.
</action>
<acceptance_criteria>
- `parser/kmz-defaults.js` exports `DEFAULT_OPTIONS`, `PRESET_COLORS`, `mergeOptions`, `resolveStyleColors`
- `mergeOptions({ lineWidth: 5 }).lineWidth === 5` and retains other defaults
- `resolveStyleColors(mergeOptions({ lineColor: 'red' })).lineColorKml === 'ff0000ff'`
- `DEFAULT_OPTIONS.iconHref` contains `placemark_square.png`
- Named exports only
</acceptance_criteria>
</task>
