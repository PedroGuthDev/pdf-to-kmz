## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Global masked getTextContent for post “inside circle” | Matches first isolated digit from **any** page string near centroid → false positives (`75`, `80`). | blocking | Only masked digits on **OCG-correlated** `textoItems` for inside refine; keep `getTextContent` whole-item as optional signal only. |
| Literal `TEXTO` only for route digits | Palhoça uses **`Texto_3`**, **`TEXTO_80`**, **`txt_moldura_intelig`**; route ops stay 0 if filter is too narrow. | blocking | Alias route sequential layer check to `isSequentialPostNumberLayerName` (exclude Poste only). |
