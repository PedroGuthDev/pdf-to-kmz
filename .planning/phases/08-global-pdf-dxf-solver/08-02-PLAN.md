The topology gate has two checks: (1) arc-order monotonicity per-branch-segment (D-10) — arc-position
must increase monotonically only WITHIN each linear run between junctions, resetting the accumulator
at each junction defined by the Phase-7 junction ground-truth; (2) hub-degree matching (D-11) — the
PDF post's authoritative-edge degree-class (1=endpoint / 2=through / ≥3=hub, computed from
source-tagged authoritative edges only, NEVER inferred-label phantoms per 07 D-15) must equal the
assigned DXF node's cable-degree-class from the adjacency graph. The accept bar then requires ALL
THREE (D-05): residual gate returns "trust" (reusing the in-tree `applyResidualGate`, which catches
the LC 179m offset), topology gate passes, and `elapsedMs < 2000`. Any one failing → the solver
returns a demotion `{ ok:false, reason }`.

Hungarian can reproduce all distances while being topologically wrong; the topology gate plus the
anchor sub-score close that gap. No cascade wiring yet (Wave 3) — validated on synthetic fixtures.
