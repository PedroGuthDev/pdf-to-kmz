## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser origin → IndexedDB | Persisted data (DXF blob + parsed posts) is same-origin; no cross-origin exposure |
| User-provided GPS → region lookup | Untrusted lat/lon input; bbox check is the only consumer — no SQL injection risk |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-10-01 | Information Disclosure | IndexedDB: region records contain user-uploaded DXF blobs | accept | Same-origin only. Personal tool, single user. No server involved (D-DWG-STORAGE-01). |
| T-02-10-02 | Tampering | coordinate-calculator-dwg.js: DWG result substituted for PDF result when DWG available | mitigate | Strict pairing (D-DWG-PAIR-01): all-or-nothing.
