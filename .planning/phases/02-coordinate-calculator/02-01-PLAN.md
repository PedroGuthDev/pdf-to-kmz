<action>
Create new ESM module parser/coordinate-calculator.js with named exports only.

Export function calculateCoordinates(posts, distances, startLat, startLon):
  1. Sort posts by number ascending.
  2. Assign startLat/startLon to post #1 (lowest-numbered post, per D-14).
  3. For each sequential pair (N, N+1):
     a. Calculate bearing from PDF coords. After flipY: +x = east, +y = south (y increases downward).
        dx = next.x - curr.x (east component).
        northward = curr.y - next.y (north component -- smaller y = further north).
        Convert to compass degrees: bearingDeg = (Math.atan2(dx, northward) * 180 / Math.PI + 360) % 360.
        WARNING: Do NOT double-negate. The correct formula is atan2(dx, curr.y - next.y).
        If you define dy = -(next.y - curr.y), then use atan2(dx, dy) -- NOT atan2(dx, -dy).
        Verify: post directly north (same x, smaller y) gives atan2(0, positive) = 0 degrees. CORRECT.
     b. Look up meters from distances[] (matching from===curr.number, to===next.number). If no distance found, use null (gap -- handled in Plan 02-02).
     c. If meters is not null, project GPS using flat-Earth approximation (D-05):
        - dLat = (meters * Math.cos(bearingRad)) / 111320
        - dLon = (meters * Math.sin(bearingRad)) / (111320 * Math.cos(lat * Math.PI / 180))
        - next.lat = curr.lat + dLat, next.lon = curr.lon + dLon
     d. If meters is null, leave lat/lon undefined for now (Plan 02-02 handles gaps).
  4. Return enriched posts[] with lat and lon fields added (D-16).

Export function parseCoordinateInput(input):
  Parse decimal degree strings like "-27.645312, -48.671234" (D-13). Support Google Maps paste format (comma-separated, optional spaces). Return { lat, lon } or null if invalid.

Export function validateBrazilBounds(lat, lon):
  Return { valid: true } if lat in [-34, 5] and lon in [-74, -35], else { valid: false, message: "Coordinates outside Brazil bounds" } (D-15). Warning only, not rejection.

Follow ESM named-export pattern, no default export, no CommonJS (established pattern from parser/).
</action>
